FreeCodeCamp - Image Search API

Endpoints:

  • Search images by term

    /api/imagesearch/:term
    (example: /api/imagesearch/lolcats)

  • Search images by term (with offset)

    /api/imagesearch/:term?offset=n
    (example: /api/imagesearch/lolcats?offset=2)

  • List of most recently submitted search strings

    /api/latest/imagesearch
    (example: /api/latest/imagesearch)
